# Futuristic Space Portfolio

A cinematic, premium, sci-fi themed developer portfolio for a Senior Flutter + Full Stack Web Engineer.

## Stack

- Next.js 15 (App Router)
- React + TypeScript
- Tailwind CSS
- Framer Motion
- GSAP + ScrollTrigger
- Three.js / React Three Fiber
- Lenis smooth scrolling
- Shadcn-style reusable UI primitives
- Lucide icons

## Experience Highlights

- Immersive 3D hero planet scene and cinematic loading intro
- Animated deep-space background with nebula gradients and particles
- Holographic projects with hover tilt physics and motion-driven previews
- Mission profile section with orbiting skill system and animated counters
- Embedded mini-game: "Protect the production server from bugs"
- Command palette, terminal mode, AI orb assistant, and Konami easter egg
- Responsive mobile-first interactions with reduced GPU pressure on small screens
